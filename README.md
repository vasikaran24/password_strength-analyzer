
# 🔐 Password Strength Analyzer

A simple Python-based cybersecurity tool that evaluates password strength and provides recommendations to improve it.

## 🚀 Features
- Checks password strength (Very Weak → Very Strong)
- Validates:
  - Length
  - Uppercase & lowercase letters
  - Numbers
  - Special characters
- Detects common weak patterns
- Provides improvement suggestions
- Suggests strong passwords

## 🛠️ Technologies Used
- Python 3
- Regex

## ▶️ How to Run

```bash
python password_analyzer.py
